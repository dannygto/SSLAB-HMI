基础信息

服务器信息

基础URL： http://localhost:8080

API前缀： /api

数据格式： JSON

字符编码： UTF-8

状态码说明

200 请求成功

400 请求参数错误

404 资源不存在

500 服务器内部错误

设备管理API

GET 获取所有设备列表
GET /api/devices

描述： 获取系统中所有虚拟设备的列表

响应示例：

[ { "id": "device-001", "name": "环境监测装置-001", "type": "ENVIRONMENT_MONITOR", "location": "实验室A", "status": "online", "lastSeen": "2025-01-04T10:30:00.000Z", "data": { "temperature": 22.5, "humidity": 45.2, "co2": 410, "pm25": 12 } } ]
GET 获取指定设备信息
GET /api/devices/{deviceId}

描述： 根据设备ID获取特定设备的详细信息

参数 类型 必需 描述
deviceId string 是 设备唯一标识符
PUT 更新设备信息
PUT /api/devices/{deviceId}

描述： 更新指定设备的配置信息

请求体示例：

{ "name": "环境监测装置-001-更新", "location": "实验室B", "config": { "sampleRate": 5000, "alertThreshold": { "temperature": 30, "humidity": 70 } } }
POST 控制设备操作
POST /api/devices/{deviceId}/control

描述： 向设备发送控制指令

请求体示例：

{ "action": "toggle_power", "parameters": { "outlet": "A1", "state": true } }

分组管理API

GET 获取所有设备分组
GET /api/groups

描述： 获取系统中所有设备分组的列表

响应示例：

[ { "id": "group-001", "name": "实验室A设备组", "description": "实验室A区域的所有设备", "devices": ["device-001", "device-002"], "createdAt": "2025-01-04T10:00:00.000Z" } ]
POST 创建新分组
POST /api/groups

请求体示例：

{ "name": "新设备分组", "description": "分组描述", "devices": ["device-001", "device-002"] }

互动教学 - 会话管理API

POST 创建新教学会话
POST /api/interactive/sessions

描述： 创建一个新的互动教学会话

请求体示例：

{ "name": "数学课堂测试", "description": "第三章代数测试", "teacherId": "teacher-001", "settings": { "maxStudents": 16, "allowLateJoin": true } }

响应示例：

{ "success": true, "session": { "id": "session-12345", "name": "数学课堂测试", "status": "active", "createdAt": "2025-01-04T10:30:00.000Z" } }
GET 获取当前活动会话
GET /api/interactive/sessions/current

描述： 获取当前正在进行的教学会话信息

DELETE 结束当前会话
DELETE /api/interactive/sessions/current

描述： 结束当前正在进行的教学会话

POST 清除会话所有数据
POST /api/interactive/sessions/clear

描述： 清除当前会话的所有数据，包括学生信息、答题记录等

互动教学 - 题目管理API

POST 创建新题目
POST /api/interactive/questions

请求体示例：

{ "text": "1 + 1 等于多少？", "type": "single", "options": { "A": "1", "B": "2", "C": "3", "D": "4" }, "correctAnswer": "B", "timeLimit": 60 }
POST 发布题目开始答题
POST /api/interactive/questions/publish

描述： 将最新创建的题目发布给学生，开始答题倒计时

GET 获取当前发布的题目
GET /api/interactive/questions/current

描述： 获取当前正在进行答题的题目信息

POST 停止当前题目答题
POST /api/interactive/questions/stop

描述： 立即停止当前题目的答题，不等待时间结束

互动教学 - 学生管理API

POST 分配学生到座位
POST /api/interactive/students/assign

请求体示例：

{ "seatId": "A1", "studentId": "student-001", "name": "张三" }
POST 学生提交答案
POST /api/interactive/students/answer

请求体示例：

{ "seatId": "A1", "answer": "B", "answerTime": 30 }
GET 获取所有学生信息
GET /api/interactive/students

响应示例：

[ { "seatId": "A1", "studentId": "student-001", "name": "张三", "status": "answered", "lastAnswer": { "questionId": "q-001", "answer": "B", "isCorrect": true, "answerTime": 30, "submittedAt": "2025-01-04T10:35:00.000Z" } } ]
POST 清除所有学生答案
POST /api/interactive/students/clear-answers

描述： 清除所有学生的答题记录，重置答题状态

互动教学 - 统计分析API

GET 获取答题统计
GET /api/interactive/statistics/answers

描述： 获取当前题目的详细答题统计信息

响应示例：

{ "questionId": "q-001", "totalStudents": 16, "answeredCount": 12, "answerDistribution": { "A": 2, "B": 8, "C": 1, "D": 1 }, "correctCount": 8, "averageTime": 35.5 }
GET 获取参与率
GET /api/interactive/statistics/participation-rate

响应示例：

{ "participationRate": 75.0, "totalStudents": 16, "participatedStudents": 12 }
GET 获取正确率
GET /api/interactive/statistics/correct-rate

响应示例：

{ "correctRate": 66.7, "totalAnswered": 12, "correctAnswers": 8 }
GET 获取平均答题时间
GET /api/interactive/statistics/average-time

响应示例：

{ "averageTime": 35.5, "minTime": 15, "maxTime": 58, "totalAnswered": 12 }

WebSocket 实时通信

WS WebSocket连接
WS /

描述： 建立WebSocket连接以接收实时设备数据和互动教学状态更新

消息格式示例：

{ "type": "deviceUpdate", "deviceId": "device-001", "data": { "temperature": 23.1, "humidity": 46.5 }, "timestamp": "2025-01-04T10:40:00.000Z" }

互动教学消息示例：

{ "type": "studentAnswer", "seatId": "A1", "studentId": "student-001", "answer": "B", "isCorrect": true, "timestamp": "2025-01-04T10:45:00.000Z" }

错误响应格式

标准错误响应

格式：

{ "success": false, "error": { "code": "VALIDATION_ERROR", "message": "请求参数验证失败", "details": { "field": "name", "reason": "名称不能为空" } }, "timestamp": "2025-01-04T10:50:00.000Z" }
常见错误代码
错误代码 HTTP状态 描述
VALIDATION_ERROR 400 请求参数验证失败
DEVICE_NOT_FOUND 404 设备不存在
SESSION_NOT_FOUND 404 会话不存在
QUESTION_NOT_ACTIVE 400 当前没有活动的题目
SEAT_OCCUPIED 400 座位已被占用
INTERNAL_ERROR 500 服务器内部错误
